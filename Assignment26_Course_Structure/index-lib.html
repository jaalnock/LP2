PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics-I - Basics of calculus and linear algebra.
  • Engineering Physics - Mechanics, optics, and modern physics.
  • Engineering Chemistry - Basic chemical principles and applications.
  • Fundamentals of Programming - Introduction to C programming.
  • Engineering Graphics - 2D/3D design and drawing skills.

Second Year (SE)

  • Data Structures - Stacks, queues, trees, and graphs.
  • Digital Logic Design - Logic gates, combinational and sequential circuits.
  • Computer Organization - CPU, memory, and assembly language.
  • Discrete Mathematics - Logic, relations, graph theory.
  • Object-Oriented Programming - Java and OOP principles.

Third Year (TE)

  • Operating Systems - Process, memory, file systems.
  • Database Management Systems - SQL, normalization, indexing.
  • Software Engineering - SDLC, testing, and project management.
  • Computer Networks - TCP/IP, routing, network layers.
  • Theory of Computation - Automata theory and formal languages.

Final Year (BE)

  • Machine Learning - Supervised, unsupervised algorithms.
  • Cloud Computing - Virtualization, services, cloud models.
  • Cyber Security - Threats, encryption, and network defense.
  • Project Work - Final year real-world project implementation.
  • Electives - IoT, Blockchain, AI, Data Mining, etc.
